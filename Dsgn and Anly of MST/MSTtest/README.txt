MSTtest : It's take random input

->Introduction:-
-The program allows users to choose how many vertices they want to give input for both Kruskal and Prim's algorithm. 
-It prompts users to select option and based on that it's run both kruskal and prim's algorithm.
-After execution, it displays the selected edges and the cost of the minimum spanning tree and time taken by that both algorithms. It's also compare these two algorithm.


->To Run the MSTtest.java file:-
-Ensure you have Java Development Kit (JDK) installed on your system.
-Compile the Java file using the following command: javac MSTtest.java
-Run the compiled program using the following command: java MSTtest.java
-Give the input of one of the display options.
